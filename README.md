# Roman_IA
Ронжин Роман 

## Описание

Приложение для поиска пользователей GitHub с возможностью добавления избранных пользователей в локальное хранилище.

## Функциональность

* Поиск пользователей GitHub по имени.
* Отображение основных данных: имя, местоположение, количество публичных репозиториев.
* Добавление пользователей в избранное.
* Сохранение избранных пользователей в JSON‑файл.
* Проверка корректности ввода.

## Использование API GitHub

Приложение использует официальный GitHub API (https://api.github.com). Для поиска пользователя отправляется GET‑запрос к эндпоинту
